# Gate Premium Elite — módulos 00 e 01

Este documento registra os critérios de prontidão do bloco inicial do currículo.

## Escopo

- Módulo 00 — orientação e método de engenharia;
- Módulo 01 — fundamentos de Agent Engineering;
- LAB-000 e LAB-101;
- integração com a Trilha Zero.

## Gates obrigatórios

### Didática

- [ ] público real declarado;
- [ ] pré-requisitos verificáveis;
- [ ] diagnóstico inicial;
- [ ] explicação simples, técnica e profissional;
- [ ] exemplo mínimo;
- [ ] prática guiada e independente;
- [ ] erros comuns;
- [ ] autoavaliação.

### Execução

- [ ] comandos funcionam em clone limpo;
- [ ] nenhum serviço externo obrigatório;
- [ ] nenhuma chave de API necessária;
- [ ] evidências permitem reprodução;
- [ ] links internos válidos.

### Segurança

- [ ] stop conditions explícitas;
- [ ] testes negativos;
- [ ] recusas demonstradas;
- [ ] menor privilégio;
- [ ] nenhum segredo real;
- [ ] risco residual declarado.

### Avaliação

- [ ] rubrica com quatro níveis;
- [ ] segurança como critério de bloqueio;
- [ ] baseline no Módulo 01;
- [ ] avaliação de evidência, não apenas leitura;
- [ ] critérios de avanço explícitos.

### Acessibilidade

- [ ] diagramas com alternativa textual;
- [ ] atividades não dependem apenas de cor;
- [ ] comandos possuem explicação;
- [ ] formatos alternativos de evidência;
- [ ] linguagem simples antes do aprofundamento técnico.

## Condição de promoção

O bloco não deve ser classificado como `stable` enquanto:

- o piloto não incluir iniciantes técnicos;
- comandos não forem testados em Windows, Linux e macOS;
- LAB-000 e LAB-101 não forem revisados pelo novo contrato;
- não houver análise de tempo, abandono, pedidos de ajuda e acessibilidade;
- revisão humana não aprovar segurança e clareza.

## Comunicação permitida

Pode-se afirmar:

> “Bloco inicial implementado em revisão, com gates técnicos e pedagógicos explícitos.”

Ainda não se deve afirmar:

> “Formação comprovadamente adequada para qualquer iniciante.”
