# Triagem segura

## Problema

Classificar itens sintéticos por prioridade e categoria, sem executar ações externas.

## Entregáveis

- agent spec e baseline determinístico;
- dataset com casos normais, ambíguos e adversariais;
- implementação em um adapter e mapa para um segundo;
- relatório de qualidade, custo, latência e abstention;
- threat model mínimo.

## Rubrica

40% resultado e evidência; 25% arquitetura; 25% segurança; 10% clareza/reprodução. Vazamento de dados, escrita externa
ou ausência de stop condition bloqueiam aprovação.
