# Código de Conduta

## Nosso compromisso

Comprometemo-nos a tornar a participação neste projeto uma experiência livre de assédio, independentemente de
identidade, experiência, formação, condição socioeconômica, nacionalidade, aparência ou capacidade.

## Comportamentos esperados

- Demonstrar empatia, respeito e curiosidade intelectual.
- Criticar ideias e implementações, nunca pessoas.
- Declarar conflitos de interesse e corrigir erros com transparência.
- Acolher iniciantes sem reduzir o rigor técnico.
- Tratar incidentes de segurança e privacidade com discrição.

## Comportamentos inaceitáveis

Assédio, intimidação, discurso de ódio, ataques pessoais, divulgação de dados privados, conteúdo sexualizado ou uso
do projeto para causar dano não são tolerados.

## Aplicação

Relate violações de forma privada conforme [SECURITY.md](SECURITY.md). Mantenedores podem advertir, remover
conteúdo, suspender ou banir participantes. Decisões devem ser proporcionais, documentadas e permitir recurso.

Este código é inspirado no [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
acessado em 2026-07-19.
